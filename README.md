# ACCA APM Master Framework

Ultimate interactive study hub for ACCA Advanced Performance Management (APM).

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main HTML structure |
| `styles.css` | All styling (dark theme, responsive layout) |
| `script.js` | Search, accordions, tabs, modals |

## How to Host on GitHub Pages

1. Create a new GitHub repository
2. Upload all three files: `index.html`, `styles.css`, `script.js`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at `https://yourusername.github.io/repo-name`

## Features

- 🔍 Instant search (press `/` to focus)
- 15 topic sections with expandable accordions
- Tabbed content per topic
- Quick tool modals (Crash Notes, Command Verbs, Exam Traps, Model Selector)
- Balanced Scorecard visual map, Porter's 5 Forces grid, decision trees
- SMART KPI examples, exam answer frameworks
- Fully responsive (mobile-friendly)
- No dependencies — works offline once loaded
